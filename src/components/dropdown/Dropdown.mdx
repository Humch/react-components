# Dropdown

**An interactive dropdown menu for discoverable content.**

## Props

| Property     | Type        | Required | Default value | Description                                         |
| ------------ | ----------- | -------- | ------------- | --------------------------------------------------- |
| children     | ReactNode   | yes      | N/A           | The content of the box                              |
| title        | string      | yes      | N/A           | The content of the dropdown button                  |
| name         | string      | no       | 'dropdown'    | The name of the aria-controls attribute             |
| isActive     | boolean     | no       | N/A           | If true, the dropdown menu will show up             |
| isHoverable  | boolean     | no       | N/A           | The dropdown menu opens on hover                    |
| isRight      | boolean     | no       | N/A           | The dropdown menu is aligned to right               |
| isUp         | boolean     | no       | N/A           | The dropdown menu appears above the dropdown button |
| buttonConfig | ButtonProps | no       | N/A           | All the props available for a Button Component      |
| dropdownIcon | ReactNode   | no       | N/A           | Replace the small caret with a custom icon          |
| className    | string      | no       | N/A           | Add custom design to the component                  |
