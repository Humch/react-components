# Block

**The block element is a simple spacer tool. It allows sibling HTML elements to have a consistent margin between them**

## Props

| Property  | Type      | Required | Default value | Description                        |
| --------- | --------- | -------- | ------------- | ---------------------------------- |
| children  | ReactNode | yes      | N/A           | The content of the block           |
| className | string    | no       | N/A           | Add custom design to the component |
