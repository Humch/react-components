# Help

**A simple paragraph HTML tag to add help text to form field**

## Props

| Property  | Type   | Required | Default value | Description                        |
| --------- | ------ | -------- | ------------- | ---------------------------------- |
| text      | string | yes      | N/A           | The text of the component          |
| color     | Colors | no       | N/A           | Change the color of the text       |
| className | string | no       | N/A           | Add custom design to the component |
