# Tabs

**Simple responsive horizontal navigation tabs, with different styles**

## Props

| Property        | Type      | Required | Default value | Description                               |
| --------------- | --------- | -------- | ------------- | ----------------------------------------- |
| children        | ReactNode | yes      | N/A           | The content of the tabs                   |
| alignment       | Alignment | no       | N/A           | The alignment of the tabs                 |
| size            | Sizes     | no       | N/A           | The size of the tabs                      |
| isBoxed         | boolean   | no       | N/A           | Add borders to the component              |
| isToggle        | boolean   | no       | N/A           | Add button radios design to the component |
| isToggleRounded | boolean   | no       | N/A           | Add rounded style to the component        |
| isFullWidth     | boolean   | no       | N/A           | Component take the whole width            |
| className       | string    | no       | N/A           | Add custom design to the component        |
