# Checkbox

**The 2-state checkbox in its native format.**

## Props

| Property       | Type      | Required | Default value | Description                                       |
| -------------- | --------- | -------- | ------------- | ------------------------------------------------- |
| children       | ReactNode | yes      | N/A           | The content of the box                            |
| isDisabled     | boolean   | no       | N/A           | Set the checkbox input data attribute to disabled |
| className      | string    | no       | N/A           | Add custom design to the input checkbox           |
| labelClassName | string    | no       | N/A           | Add custom design to the label                    |
