# ProgressBar

**Native HTML progress bars**

## Props

| Property  | Type   | Required | Default value | Description                        |
| --------- | ------ | -------- | ------------- | ---------------------------------- |
| value     | number | no       | N/A           | The value of the progress bar      |
| max       | number | no       | N/A           | The max of the progress bar        |
| color     | Colors | no       | N/A           | The color of the progress bar      |
| size      | Sizes  | no       | N/A           | The size of the progress bar       |
| className | string | no       | N/A           | Add custom design to the component |
