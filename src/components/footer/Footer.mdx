# Footer

**A simple responsive footer which can include anything: lists, headings, columns, icons, buttons, etc.**

## Props

| Property  | Type      | Required | Default value | Description                        |
| --------- | --------- | -------- | ------------- | ---------------------------------- |
| children  | ReactNode | yes      | N/A           | The content of the footer          |
| className | string    | no       | N/A           | Add custom design to the component |
