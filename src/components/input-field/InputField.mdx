# InputField

**The text input and its variations**

## Props

| Property          | Type       | Required | Default value | Description                        |
| ----------------- | ---------- | -------- | ------------- | ---------------------------------- |
| name              | string     | yes      | N/A           | The name property of the input tag |
| label             | ReactNode  | yes      | N/A           | The content of the label           |
| children          | ReactNode  | no       | N/A           | Add content under the input field  |
| color             | Colors     | no       | N/A           | Change the color of the component  |
| size              | Sizes      | no       | N/A           | Change the size of the component   |
| labelProps        | LabelType  | no       | N/A           | Add custom props to the label      |
| inputProps        | InputType  | no       | N/A           | Add custom props to the input tag  |
| leftIcon          | IconsProps | no       | N/A           | Add custom icon to the left        |
| rightIcon         | IconsProps | no       | N/A           | Add custom icon to the right       |
| controlIsExpanded | boolean    | no       | N/A           | Expand control component           |
